* [tpm](https://github.com/tmux-plugins/tpm) for tmux plugins 
* [kunst](https://github.com/sdushantha/kunst) & [sxiv](https://github.com/muennich/sxiv) for Mpd cover 
* the scripts needed are in "./scripts"
